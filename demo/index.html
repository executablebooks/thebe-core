thebe-core: available demos

This is a listing of the available demos for thebe-core in use from the native js api,

  • Basic demo connecting to mybinder.org
  • ipywidgets demo connecting to mybinder.org
  • ipywidgets demo connecting to a local Jupyter instance
  • ipyleaflet demo connecting to a local Jupyter instance
  • Demo of rendering a single cell without use of the Notebook interface
  • Demo of rendering a single cell without use of the Notebook interface
  • Demo of different kernel controls and interaction options
  • Demo of running against a JupyterLite kernel
